Exploring NAMELIST

The NAMELIST features of Fortran have a number of sometimes surprising uses such as reading command line arguments, processing configuration files, easily creating problem sets from input files, and even providing for interactive input.

  • NAMELIST-based parsing of command line arguments
  • interactively editing variables names via a NAMELIST group
  • Using NAMELIST I/O for configuration files
  • Capture and Replay for unit testing
  • basic usage with multiple cases in a file
  • snapshotting the current state for checkpointing, state restore, ...
  • Using ISO_C_BINDING interface to read NAMELIST from C by calling Fortran
